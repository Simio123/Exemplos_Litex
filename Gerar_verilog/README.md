# Exemplo Migen: Somador de 8 Bits

Este exemplo mostra como utilizar o **Migen** para descrever um circuito digital simples em Python. O circuito implementa um **somador de 8 bits** e, ao executar o script, o Migen converte automaticamente a descrição para **Verilog**, gerando o arquivo `somador.v`.

---

## 📄 Arquivo

- `somador.py`: Descreve o circuito em Migen e gera o código Verilog.

---

## ▶️ Como Executar

Execute o script:

```bash
python3 somador.py
```

Ao final da execução, será gerado o arquivo:

```text
somador.v
```

contendo a implementação do somador em Verilog.
